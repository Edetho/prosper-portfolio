## Pour visualiser le portfolio localement

1. **Ouvrir directement** : Double-cliquez sur `index.html` dans l'explorateur
2. **Avec un serveur local** (recommandé pour éviter les restrictions CORS) :
   - **Python** : `python -m http.server 8000`
   - **Node.js** : `npx serve .`
   - Puis ouvrez http://localhost:8000
